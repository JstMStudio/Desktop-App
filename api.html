API

JstMStudio provided a tons of services in our software and websites like in JstMStudio Store and JstMStudio App. JstMStudio API is an Application Programming Interface of JstMStudio. JstMStudio API provide support for all developer to implement JstMStudio services on our website and software onto your website without developing the features itself and for free. All documentation here is retrieved from the latest desktop update from JstMStudio that had been downloaded in this device. Please download the latest desktop app version from JstMStudio to get the latest API documentation



Open Source

JstMStudio API is an open source API ( Application Programming Interface ). Everyone could view, read and study all of JstMStudio API source codes on our official Github repository. JstMStudio API is built using HTML, CSS and vanilla JavaScript unlike any traditional API too and licensed under MIT.



Free

JstMStudio API is free for everyone to use and enjoy. Everyone could use JstMStudio API to implement services provided by JstMStudio for free with no hidden fees. JstMStudio API is built for everyone to use and not for earning money from developers and website owner.
